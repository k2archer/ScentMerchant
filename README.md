# ScentMerchant
### 项目描述

**What is KShare?**

仿广告扫码平台商家端，实体店铺进驻、扫码激活投放、统计收益。

本 App 整体用 MVP 实现，目前实现 14 个页面，大部份使用约束布局完成。

### 第三方库

* BufferKnife
