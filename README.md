# ScentMerchant
### 项目描述

**What is KShare?**

仿扫码广告平台商家端，实体店铺进驻、扫码激活投放、统计收益。

本 App 整体用 MVP 实现，目前实现 14 个页面，大部份使用约束布局完成。
使用 RxJava2 + Retrofit2 进行 Http API 数据请求 。

### 第三方库

* BufferKnife
* RxJava2
* Retrofit2
* Gson
